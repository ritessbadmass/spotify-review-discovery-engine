# Offline Data Acquisition Scripts

This directory contains offline scripts designed to programmatically fetch, filter, and normalize real user feedback from external platforms (Google Play, App Store, and Reddit).

These scripts are strictly decoupled from the Next.js web application. They output CSV files into the `/data` folder, which can then be manually uploaded into the Spotify Review Discovery Engine via the `/ingest` web interface.

## Prerequisites

1. **Node.js environment**: Ensure you have run `npm install` in the project root.
2. **Reddit API Credentials (Optional but recommended)**:
   If you wish to run the Reddit scraper, you must create an App at https://www.reddit.com/prefs/apps and create a `.env` file in the root of the project with the following keys:
   ```env
   REDDIT_USER_AGENT="node:spotify-discovery-scraper:v1.0 (by /u/YOUR_USERNAME)"
   REDDIT_CLIENT_ID="your_client_id"
   REDDIT_CLIENT_SECRET="your_client_secret"
   ```

## Available Commands

From the root of the project, you can run:

- `npm run scrape:play` 
  - Fetches up to 300 recent reviews from the Google Play Store (`com.spotify.music`).
- `npm run scrape:appstore`
  - Fetches recent reviews from the iOS App Store.
- `npm run scrape:reddit`
  - Fetches top posts and comments from `r/spotify` and `r/truespotify`. Gracefully skips if `.env` is missing.
- `npm run scrape:all`
  - Sequentially runs all three scrapers above and then merges them into a single `scraped-reviews.csv`, deduplicating across sources.

## Known Limitations

- **Apple App Store Volume**: The App Store scraper API relies on RSS feeds that Apple aggressively limits. You will rarely get more than 50-100 reviews per run.
- **Reddit API Limits**: Snoowrap is configured to fetch a conservative number of hot posts to avoid immediate rate limiting. 
- **Deterministic Hashing**: Usernames are hashed to pseudonyms (e.g., `user_8f4a1`) to protect user privacy.

## Importing Data into the App

After running `npm run scrape:all`:
1. Start the web app (`npm run dev`).
2. Navigate to `http://localhost:3000/ingest`.
3. Choose "Upload CSV" and select the `data/scraped-reviews.csv` file generated by these scripts.
